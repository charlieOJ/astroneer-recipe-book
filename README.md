# Astroneer Recipe Book

This project was create after hours on the game [Astroneer](https://astroneer.space/), looking for item to craft and how to craft the resources for it.

It was create with [Create React App](https://github.com/facebook/create-react-app), using the [TypeScript](https://www.typescriptlang.org/) template. It use [React-router](https://reactrouter.com/) to handle multiple pages, [i18next](https://www.i18next.com/) for translations and [Bootstrap 5.3](https://getbootstrap.com/docs/5.3/getting-started/introduction/).

## Available Scripts
